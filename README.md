Install Vundle first, then run +PluginInstall.
For the color scheme, create directory ~/.vim/colors and move ~/.vim/bundle/vim-colors-solarized/colors/solarized.vim to ~/.vim/colors.

## plugins
* delimitMate
* vim-indent-guides
* lightline
* ctrp-p
* vim-eunuch
* vim-surround
* nerdtree (binded to ctrl-O)
* vim-gitgutter
* emmet-vim
* vim-javascript
* vim-jsx
* ale

## tab settings
* plugin indent on
* tabstop = 4
* shiftwidth = 4
* expandtab
